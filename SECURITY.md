# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this app, please report it responsibly by emailing the maintainer directly rather than opening a public issue.

## Scope

This app communicates with your local MQTT broker and Tasmota devices on your home network. It does not connect to any external cloud services. MQTT credentials are stored locally on your Homey device via Homey's built-in settings storage.
